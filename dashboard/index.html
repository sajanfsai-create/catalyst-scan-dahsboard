📋 What is a Device Fingerprint?

A Device Fingerprint is a unique SHA-256 hash automatically generated by the CatalystScan .exe when it runs on a machine. The user does NOT need to enter or collect anything.

Motherboard Serial + CPU Processor ID + Primary Disk Serial
Windows reinstall, .exe deletion, new user accounts
Uniquely identifies a physical machine for licensing & anti-theft

⚙️ How the .exe Generation Works

1 Enter your organization name, ID, and server URL below
2 Click "Generate .exe Package" — the system creates a .zip containing:
• CatalystScan.exe — The scanner application
• org_config.json — Your org identity baked in
• README.txt — Instructions for deployment
3 Download the .zip and distribute to your lab PCs
4 When the .exe runs, it auto-registers with this dashboard using the embedded org config
